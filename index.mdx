Outcome provides the `@outcomexyz/hip4-sdk` — a fully typed TypeScript SDK for integrating with HIP-4 prediction markets on Hyperliquid. Whether you're building a trading interface, analytics dashboard, or automated strategy, the SDK gives you everything you need: market discovery, real-time WebSocket subscriptions, order placement with EIP-712 signing, and USDH fund management — all with zero runtime dependencies.

## Get started in 4 steps

<Steps>
  <Step title="Install the SDK">
    Add `@outcomexyz/hip4-sdk` to your project with your package manager of choice.
    ```bash
    pnpm add @outcomexyz/hip4-sdk
    ```
  </Step>
  <Step title="Create an adapter">
    Instantiate the HIP-4 adapter and initialize it to warm up the event cache.
    ```typescript
    import { createHIP4Adapter } from "@outcomexyz/hip4-sdk";

    const hip4 = createHIP4Adapter({ testnet: true });
    await hip4.initialize();
    ```
  </Step>
  <Step title="Fetch markets">
    Query live prediction markets with filtering, grouping, and pagination.
    ```typescript
    const markets = await hip4.events.fetchMarkets({ type: "defaultBinary" });
    console.log(markets[0].underlying); // "BTC"
    ```
  </Step>
  <Step title="Authenticate and trade">
    Approve an agent key, then place limit and market orders silently.
    ```typescript
    await hip4.auth.initAuth(userAddress, agentSigner);
    const result = await hip4.trading.placeOrder({
      marketId: "516",
      outcome: "#5160",
      side: "buy",
      type: "limit",
      price: "0.65",
      amount: "100",
    });
    ```
  </Step>
</Steps>
