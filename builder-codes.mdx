To trade on Outcome, you'll need to add Outcome's builder code to your Hyperliquid account. This is a one-time setup step prompted during your first sign-up to [outcome.xyz](http://outcome.xyz).

### What a builder code is

A builder code is a Hyperliquid account-level setting that authorizes a frontend like Outcome to facilitate your trades. Once added, any trade you place through Outcome will include the builder code, which allows Outcome to receive a portion of the trading fee as compensation for running the interface.

Builder codes are mandatory for trading on Outcome. Without one attached to your account, your trades cannot be placed through [outcome.xyz](http://outcome.xyz).

### How to add it

Outcome's interface prompts you to authorize the builder code the first time you sign up. You'll see a transaction request to approve. Once approved, no further action is needed; the code stays attached to your account.

### Outcome's builder code address

```text
0xab5dbc057628bc18523c4cdfc0e1e2ebdbecb704
```

You can verify the builder code attached to your account from your Hyperliquid account settings at any time.

### Fee implications

The builder code can charge a fee on top of Hyperliquid's standard trading fees, which goes to Outcome. The builder code fee is currently set to zero. This may change in the future. See Fees for the full fee structure.
